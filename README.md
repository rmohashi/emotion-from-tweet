## Setup

1. Install the dependencies (use ***requirements.gpu.txt*** instead of
*requirements.txt* when using GPU processing):

    ```bash
    pip install -r requirements.txt
    ```
1. Create a `.env` file:

    ```bash
    cp .env.sample .env
    ```
1. Set the environment variables inside the created `.env` file
